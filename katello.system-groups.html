katello.system-groups documentation

add-to
(add-to group system)
Adds a system to a system group

copy
(copy orig-name new-name & [{:keys [description]}])
Clones a system group, given the name of the original system group
to clone, and the new name and description.
create
(create name & [{:keys [description]}])
Creates a system group

edit
(edit sg-name {:keys [new-limit new-sg-name description]})
Change the value of limit field in system group

remove
(remove group & [{:keys [also-remove-systems?]}])
Removes a system group. Optionally, remove all the systems in the
group as well.
remove-from
(remove-from group system)
Remove a system from a system group

system-count
(system-count sg-name)
Get number of systems in system group according to the UI
